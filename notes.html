Saturday, 8/2: Git for Everyone Workshop

Dale Sande, instructor

Git for Everyone by Dale Sande.
    General notes from class:
  • Intro to Terminal/BASH (pwd, ls -la, cd, mkdir, touch, cd, rm -rf, clear, and so on)
  • With mv or cp involving a directory, it matters if the whack is added: without the whack, the entire directory is moved; with the whack, only the contents of the directory are moved
  • There is a keystroke cheatsheet in Dale's book. LEARN THEM
  • Make aliases in your BASH profile to streamline workflow. Talk to people, ask around for suggestions. Dale's book has some.
